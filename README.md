# Landing page de Carvatel

Sitio empresarial público desarrollado con Next.js, TypeScript y Tailwind CSS.

## Ejecutar localmente

```bash
npm install
npm run dev
```

Abre `http://localhost:3000` en el navegador.

## Producción

```bash
npm run build
npm start
```

Los datos de contacto se administran desde `config/company.ts`. La página no incluye backend ni envía el formulario a ningún servicio externo.
